#### Material design - AppBarLayout, FloatingActionButton

#### AppCompatActivity
상위 안드로이드 버전에서는 지원이 되는데 하위 버전에서는 지원이 안되는 메소드가 있다.  
그런 메소드들을 하위 버전에서도 똑같이 지원할 수 있게 만드는 Support Library가 AppCompatActivity이다.
